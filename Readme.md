## Сервер для разработки

Для запуска лайв-сервера нужен [Node.js](https://nodejs.org) версии 0.12 или выше.

### Установка

```bash
$ git clone https://github.com/sedona.git
$ cd sedona
$ npm i
```

### Запуск

```bash
$ npm start
```

После выполнения команды, откройте в браузере `http://127.0.0.1:3000`.

### Тестирование

```bash
$ npm test
```
